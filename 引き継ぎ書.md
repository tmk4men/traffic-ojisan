## 4. アプリを更新するとき（超重要）

ゲーム（index.html等）を直したら、Mac で：

```bash
git pull
npm run sync        # = build:www + cap sync ios（www再生成→アプリに反映）
npx cap open ios    # Xcodeでビルド/Archive
```

Xcodeで **General の Build番号を +1**（例 1→2）してから **Product → Archive → Distribute App**。

> ⚠️ **`index.html` を直しただけではアプリに反映されない。** 必ず `npm run sync` が必要。
> （Webの公開サイトはpushだけで更新されるが、アプリはsync＋再Archiveが要る）

## 6. あとで広告（AdMob）を入れる場合

`index.html` に AdMob連携コードは実装済みだが、**プラグイン未導入なので現在は無効（no-op）**。
有効化手順（詳細は `iOS-BUILD.md` の「AdMobを後から有効化する」）：

1. **iOS用のAdMobを別途用意**（今のIDは**Android用**。iOSは別アプリ登録＋別の広告ユニットIDが必要）。
2. `npm install @capacitor-community/admob` → `npx cap sync ios`
3. `Info.plist` に `GADApplicationIdentifier`・`SKAdNetworkItems`・（必要なら）`NSUserTrackingUsageDescription` を追加。
4. ATT（トラッキング許可）を出すなら、Appプライバシー申告を「トラッキングあり」に更新。
5. `index.html` の `AD_INTERSTITIAL_ID`/`AD_BANNER_ID` を iOS用IDで出し分け。
